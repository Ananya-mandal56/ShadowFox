# Beginner Task

- Seaborn

- Introduction to Data Visualization
- Why Visualization Matters
- Matplotlib Overview
- Seaborn Overview
- Line Plot
- Scatter Plot
- Bar Chart
- Histogram
- Pie Chart
- Box Plot
- Area Chart
- Heatmap
- Comparison of Matplotlib and Seaborn

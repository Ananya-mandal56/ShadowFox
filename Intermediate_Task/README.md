# Intermediate Task - Delhi Air Quality Analysis

## Project Overview

This project was completed as part of the ShadowFox Data Science Internship.

The objective is to analyze Delhi's air pollution data using Python and visualize pollutant trends through Exploratory Data Analysis (EDA).

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Key Findings
- Conclusion
